# Яндекс.Музыка → Discord RPC

Небольшая программа, которая показывает в Discord статус «Слушает … в Яндекс.Музыке» на основе того, что играет в веб-версии Яндекс.Музыки.

## Как это работает

1. **Программа (Node.js)** — запускается на твоём компьютере, подключается к Discord по RPC и поднимает WebSocket-сервер на `localhost:8765`.
2. **Userscript в браузере** — на странице Яндекс.Музыки считывает название трека и артиста и отправляет их в программу по WebSocket.
3. Discord отображает это как активность (Rich Presence).

## Что нужно

- **Node.js** (например, LTS с [nodejs.org](https://nodejs.org))
- **Discord** установлен и запущен
- **Браузер** с расширением для userscript’ов: [Tampermonkey](https://www.tampermonkey.net/) или [Violentmonkey](https://violentmonkey.github.io/)

## 1. Создать приложение в Discord (один раз)

1. Зайди в [Discord Developer Portal](https://discord.com/developers/applications).
2. **New Application** — придумай имя (например, «Yandex Music»).
3. Открой созданное приложение → вкладка **General Information**.
4. Скопируй **Application ID** — это твой `Client ID`.

## 2. Установка и запуск программы

В папке проекта (в терминале):

```bash
cd yandex-music-rpc
npm install
```

Если `npm install` выдаёт ошибку — открой терминал от имени пользователя и выполни эти команды снова.

Запуск (Discord должен быть запущен):

```bash
npm start
```

- В коде в `index.js` замени `CLIENT_ID` на свой Application ID.
- И затем `npm start`.

В консоли должно появиться что-то вроде:

- `Discord RPC подключён.`
- `WebSocket сервер: ws://localhost:8765`

Программу не закрывай, пока хочешь видеть статус в Discord.

## 3. Userscript в браузере

1. Установи расширение **Tampermonkey** или **Violentmonkey**.
2. Открой файл `userscript.user.js` из этой папки в редакторе, скопируй весь код.
3. В расширении создай новый userscript и вставь этот код, сохрани.
4. Открой [music.yandex.ru](https://music.yandex.ru) и включи воспроизведение.

Если программа запущена и Discord открыт, через несколько секунд в профиле Discord должна появиться активность с названием трека и артистом.

## Если статус не обновляется

- Убедись, что программа запущена и в консоли нет ошибок.
- Убедись, что вкладка с Яндекс.Музыкой открыта и трек действительно играет.
 - Перезагрузи компьютер, если пишет `Discord RPC: запусти программу на ПК (npm start)`
 - Или в логах есть строки вроде: `/track clear+shutdown POST failed: Невозможно соединиться с удаленным сервером`

## Сброс статуса

- Закрой вкладку с Музыкой и подожди около минуты — программа сама сбросит активность.
- Нажми Exit в трее для полного завершения и сброса статуса.
- Нажми Stop RPC в трее для сброса статуса.

## Очистка логов

- В трее выбери `Clear logs` затем:
 - Пропиши `0` для очистки всех логов кроме последнего.
 - Или выбери `Clear all incl. last` для очистки всех логов включая последний

## Файлы

- `index.js` — основная программа (Discord RPC + WebSocket).
- `userscript.user.js` — скрипт для браузера (копировать в Tampermonkey/Violentmonkey).
- `README.md` — эта инструкция.
