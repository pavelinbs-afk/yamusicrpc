# Яндекс.Музыка → Discord

Обычное приложение для Windows: **установил — запустил**, без командной строки и без лишних окон.

**Готовые установщики** со страницы релизов на GitHub:

- **`.exe`** (NSIS) — обычный мастер установки, ярлык в меню «Пуск».
- **`.msi`** — по желанию, тот же продукт в формате Windows Installer (удобно для корпоративных сценариев).
- **`portable.exe`** — один файл, без установки в систему.

Внутри уже упакованы программа, движок Electron и нужные библиотеки. **Node.js ставить не нужно.**

## Что сделать

1. Скачай установщик `Yandex Music RPC` (`.exe`, при необходимости `.msi` или portable) со страницы релизов проекта.
2. Запусти установщик и пройди шаги (для portable — просто запусти файл).
3. Открой программу из меню **«Пуск»** или с ярлыка на рабочем столе.
4. Запусти **десктопное приложение Яндекс.Музыки** (не сайт в браузере), войди в аккаунт и включи воспроизведение.
5. Держи открытым **Discord** — в профиле появится статус «слушает …».

В окне программы сверху полоска **RPC** и строка с текущим треком; кнопка **«Настройки»** открывает разделы в том же окне (не отдельное «липкое» окно у плеера).

## Что должно быть запущено

- **Discord** (программа для Windows)
- **Яндекс.Музыка** — официальный клиент для Windows

## Настройки

После нажатия «Настройки» доступны разделы: статус, Discord, источник, внешний вид, логи. Всё на русском.

## Обложка в Discord

Для трека из десктопного клиента подставляется запасная картинка приложения. При необходимости свой рисунок можно задать в [портале разработчика Discord](https://discord.com/developers/applications) → Rich Presence → Art Assets (ключ **`yandex_music_icon`**).
