Верхняя полоска: RPC и текущий трек. Кнопка «Настройки» открывает эту панель в том же окне. Трек берётся из десктопного клиента Яндекс.Музыки (Windows). Запустите клиент и включите воспроизведение.

Интеграция Discord

Статус «Слушает…», обложка и таймер. При отключении активность в Discord сбрасывается.

Вторая кнопка в Discord ведёт на фиксированный адрес из приложения. Если выключить, остаётся только кнопка открытия трека.

Источник трека

Трек и исполнитель берутся из заголовка окна официального приложения Яндекс.Музыки (Windows). Веб-версия и встроенный браузер не используются.

Если опрос выключен, статус из клиента не подхватится. Для обложек в Discord по-прежнему нужен трек из клиента; отдельной обложки из заголовка окна нет — используется иконка приложения в Discord.

Клиент Windows не сообщает длину трека — для зелёной полосы в Discord берётся это число (по умолчанию ~3,5 мин). Поставьте 0, если полоса не нужна.

Внешний вид панели

Для десктопного клиента обложки по URL недоступны — в Discord показывается запасной asset.
